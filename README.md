# Go Task REST API

## Описание
REST API-сервис для управления задачами (task management), написанный на Go. Разработан в рамках практики по освоению разработки серверной части приложений с использованием стандартной библиотеки и фреймворка `chi`.

Функциональность включает:
- Добавление, редактирование, удаление задач
- Получение списка всех задач
- Завершение задач (done)

## Технологии
- Язык программирования: Go
- Маршрутизатор: [chi](https://github.com/go-chi/chi)
- Работа с БД: `database/sql`
- Формат обмена данными: JSON
- Структурирование: MVC-like (модели, хендлеры, роутер)
